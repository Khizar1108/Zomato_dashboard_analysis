# 📌 Project Overview

This project presents an interactive Zomato Sales Analysis Dashboard designed to explore restaurant data across multiple countries. It provides insights into restaurant distribution, pricing, cuisines, ratings, and online delivery trends, helping stakeholders make data-driven decisions.

# Dataset Description

The dataset contains information about restaurants listed on Zomato, including:
	•	Restaurant Name
	•	City & Country
	•	Cuisine Types
	•	Average Cost for Two
	•	Ratings
	•	Online Delivery Availability

# Dashboard Features

# Key Metrics
	•	Total Restaurants: 9,545
	•	Average Cost for Two: 25M (aggregated)
	•	Average Ratings Count: 55K

# Visual Insights
	•	Top Restaurants by City – Highlights popular restaurants in different cities
	•	Top Countries with Average Costs – Shows spending trends across countries
	•	Country Distribution Map – Geographic visualization of restaurant presence
	•	Top Cuisines – Most popular cuisine types based on data
	•	Online Delivery Ratings – Comparison of delivery availability (Yes/No)
	•	Price Range Filter – Dynamic filtering based on cost categories
